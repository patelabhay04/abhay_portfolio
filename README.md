# Abhay Patel — Portfolio

This repository contains a modern, responsive portfolio website for Abhay Patel.  
Design blends clean minimal layout with colorful gradient accents.

## Files
- `index.html` — main page
- `style.css` — styles and layout
- `script.js` — simple interactions
- `resume.docx` — downloadable resume (from user's uploaded DOCX). fileciteturn1file0

## Deploy
1. Create a new GitHub repo.
2. Upload these files to the root of the repo.
3. Enable GitHub Pages in Settings → Pages using Branch `main` and folder `/ (root)`.
4. Visit the provided URL.
